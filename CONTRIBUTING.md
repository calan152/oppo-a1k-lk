# Contributing

Contributions are welcome when their provenance and redistribution terms are
clear.

Do not submit:

- private or production signing keys;
- `lk.bin`, boot/recovery images, DTB/DTBO dumps, OFP packages, or partition
  dumps;
- IMEI, NVRAM, calibration, serial-number, or other device-unique data;
- proprietary prebuilt libraries or executables without redistribution terms;
- decompiled source presented as original source.

Keep original copyright and license headers. Identify code adapted from another
project in the commit message and in `THIRD_PARTY_NOTICES.md`. Prefer reviewable
source patches and reproducible generation steps over generated binaries.
